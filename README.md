# Hello World Chrome Extension

A simple "Hello World" style Chrome extension.

## Installation

1. Open Chrome and navigate to `chrome://extensions/`
2. Enable "Developer mode" (toggle in the top right)
3. Click "Load unpacked"
4. Select this folder
5. The extension icon should appear in your Chrome toolbar

## Files

- `manifest.json` - Extension configuration (Manifest V3)
- `popup.html` - The popup UI
- `popup.css` - Styling for the popup
- `popup.js` - JavaScript for the extension
- `icons/` - Extension icons (you'll need to add icon files)

## Adding Icons

To complete the setup, add icon files to the `icons/` folder:
- `icon16.png` (16x16 pixels)
- `icon48.png` (48x48 pixels)
- `icon128.png` (128x128 pixels)

You can use any image editor to create these, or use placeholder images for testing.
